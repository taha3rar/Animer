Welcome to Ecosystems!

The Ecosystems feature is where you can create customized groups of contacts with common characteristics, such as location, product type, operation type and more. To add a new ecosystem, simply start by clicking on the New Ecosystem button.

close

New Ecosystem

Adding a new ecosystem is composed of 2 simple steps:

  • Add ecosystem details
  • Choose the contacts you would like to add to the ecosystem.

At the end of each step, click on the Next button to proceed.
